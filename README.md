Parkhaus Ticket System

 Beschreibung

Dieses Projekt simuliert ein Ticketsystem für ein Parkhaus.
Wenn ein Fahrzeug einfahren möchte, wird ein Ticket erstellt mit Einfahrtszeit und Ticketnummer.
